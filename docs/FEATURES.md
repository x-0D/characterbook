# 🎭 CharacterBook - Возможности приложения

## ✨ Обзор

CharacterBook - это мощное кроссплатформенное приложение для создания и управления персонажами RPG, предлагающее полный набор инструментов для игроков и мастеров.

- **📄 PDF экспорт** - Генерация профессиональных листов персонажей
- **⚙️ Настраиваемые форматы** - Выбор включаемых полей для экспорта
- **🖨️ Печать** - Оптимизация для бумажных носителей
- **↗️ Совместное использование** - Удобная отправка мастеру или игрокам
